# 🥗 Obesity Levels Estimation & Analysis (UCI ID: 544)

Ushbu loyihada insonlarning ovqatlanish odatlari va jismoniy holati asosida ularning semirish darajasini (Obesity Level) aniqlash va tahlil qilish amalga oshirilgan. Ma'lumotlar to'plami sifatida **UCI Machine Learning Repository**dan olingan [Obesity Levels Dataset](https://archive.ics.uci.edu/dataset/544/estimation+of+obesity+levels+based+on+eating+habits+and+physical+condition) ishlatildi.

## 🎯 Loyiha Maqsadi
Asosiy maqsad — insonning kundalik odatlari (suv ichish, harakatlanish, fast-food iste'moli va h.k.) va jismoniy ko'rsatkichlari (yosh, vazn, bo'y) orqali semirish xavfini bashorat qilish hamda muhim bog'liqliklarni vizuallashtirish.

## 📊 Ma'lumotlar Tahlili (EDA)
Loyiha davomida quyidagi tahlillar o'tkazildi:
* **Target Distribution:** Semirish darajalari (NObeyesdad) bo'yicha bemorlarning taqsimoti tahlil qilindi.
* **Correlation Analysis:** Vazn va yosh o'rtasidagi bog'liqliklar vizuallashtirildi.
* **Lifestyle Factors:** Ovqatlanish va jismoniy faollik ko'rsatkichlarining o'zaro ta'siri o'rganildi.

## 🛠 Texnologiyalar
* **Python** (Google Colab muhitida)
* **Pandas:** Ma'lumotlarni manipulyatsiya qilish uchun.
* **Seaborn & Matplotlib:** Ma'lumotlarni vizuallashtirish uchun.
* **ucimlrepo:** UCI datasetlarini yuklab olish uchun.

## 🚀 Qanday foydalanish kerak?
1. Repozitoriyani klonlang: `git clone https://github.com/Siddiq/Obesity-Estimation-ML-Analysis.git`
2. Kerakli kutubxonalarni o'rnating: `pip install ucimlrepo pandas seaborn matplotlib`
3. Jupyter Notebook faylini ishga tushiring.
